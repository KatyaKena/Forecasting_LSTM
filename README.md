# Forecasting_LSTM
Прогнозирование требований к населению в разбивке по категориям заемщиков: 3 целевые переменные.
Используется 2 вида подбора гиперпараметров: смешанный, чистый.
При этом рассматривается full-dataset и отобранный на предыдущем этапе.
Затем результаты сравниваются с бенчмарками, проверяется значимость различий с помощью теста Диболда-Мариано.
